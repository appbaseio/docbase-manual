This is v2 (in the works) of the docbase manual. If you are looking to get started, check out the getting started page here.

Roadmap

Planned features in the upcoming releases:

  1. Support for sub-topics: one more level of directory nesting
  2. Support auto-infer mode for reading .md from a filesystem
  3. Adding external URLs (like apiary)
  4. Polishing: Better search navigation in mobile view, code snippet styling support, integration with more CI tools.
